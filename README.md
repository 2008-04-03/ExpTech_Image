## 簡介

此軟體結合TREM生成的圖片，並透過 GUI 介面讓使用者能夠更方便的觀看即時的地震資訊。

### 主要功能

- 總覽：顯示 `震度速報` `地震速報` `地震報告` `長週期地震動` 的圖片
- 震度速報：顯示 `震度速報` 的圖片
- 地震速報：顯示 `地震速報` 的圖片
- 地震報告：顯示 `地震報告` 的圖片
- 長週期地震動：顯示 `長週期地震動` 的圖片
- 開啟音效：當資料更新時發出音效，**當按鈕為 *開啟音效* 時，會發出音效**
